# Mixins

# 文字一行超出省略号

@include ellipsis

.ellipsis()

# 多行文字超出省略号

@include ellipsis-multiline($number)

.ellipsis-multiline(@number)

# Params

参数说明类型默认值
number设置在第几行超出显示省略号number4

# 美化文本选中

@include beauty-select($color,$bgcolor)

.beauty-select(@color,@bgcolor)

# Params

参数说明类型默认值
color设置字体颜色十六进制数、颜色#fff（白）
bgcolor设置背景颜色十六进制数、颜色#000（黑）

# 毛玻璃

@include blur($blur)

.blur(@blur)

# Params

参数说明类型默认值
blur设置虚幻像素string10px

# 滤镜

将彩色照片显示为黑白照片、保留图片层次。

@include grayscale

.grayscale()

# 移动端 100vh

@include vh($height)

.vh(@height)

# Params

参数说明类型默认值
height设置高度string100vh
