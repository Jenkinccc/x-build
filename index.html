X-BUILD

一个纯净的前端脚手架

完整学习 →

最新技术栈

以 Vue3.0 框架为基础，使用 vuex4、vue-router4、element-plus 等前端前沿技术开发。

丰富功能

根据典型的业务模型，提供了常用的组件、状态管理、函数工具、权限验证、快捷功能等功能。

严格的开发规范

默认使用 Typescript，根据项目结构、代码规范、开发规则，协助你形成良好的开发习惯。

# 快速上手

X-BUILD 是一个基于 Node.js (opens new window) 环境的命令行工具。

npm install x-build -g

目前状态是内测开发阶段，安装请查看 NPM 版本 (opens new window) 安装 alpha 最新版本。

通过以下指令创建一个新项目，并且会自动帮你安装所有依赖：

x-build create [name]
