# 当前设备

用于PC、移动等适配

## 使用

- **type** `String，当前设备类型`
  - mobile：移动
  - tablet：平板电脑
  - desktop：PC桌面
  - unknown：未知

- **orientation** `String，当前设备定位(方向)`
  - landscape：横向
  - portrait：竖立

- **os** `String，操作系统`
  - ios
  - iphone
  - ipad
  - ipod
  - android
  - blackberry
  - windows
  - macos
  - fxos
  - meego
  - television
  - unknown
